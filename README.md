# Earnapp Devices

Standalone Next.js app for loading Earnapp devices, viewing usage forecasts, and deleting devices with a single confirmation prompt.

## Run

```bash
npm install
npm run dev
```

Optional environment overrides:

```bash
EARNAPP_APP_ID=earnapp
EARNAPP_VERSION=1.634.431
EARNAPP_USER_AGENT=Mozilla/5.0
```
